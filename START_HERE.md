## üîß Troubleshooting

### "Port 3000 is already in use"
No problem! The server will automatically use port 3001, 3002, etc.

### "npm: command not found"
You need to install Node.js first: https://nodejs.org/

### "Dependencies not installed"
Run: `npm install`

### Blank screen in browser
1. Check browser console (Press F12)
2. Make sure you're at http://localhost:3000
3. Try hard refresh (Ctrl+Shift+R)

## üìä The Data Story

The mock data tells a realistic story:

- **Flagship Store Entrance**: 5,420 passersby, 87 engagement score
  - High traffic, good placement, solid performance

- **Mall Corridor Interactive**: 3,820 passersby, 92 engagement score
  - Lower traffic but HIGHEST engagement (interactive screens work!)

- **Checkout Area**: 2,180 passersby, 95 engagement score
  - BEST performer - captive audience at checkout

- **Window Display**: 8,920 passersby, 68 engagement score
  - Highest traffic but lowest engagement (need better content)

- **Food Court**: 4,560 passersby, 83 engagement score
  - Good all-around performance

**Key Insight**: Interactive screens and captive audiences (checkout) drive highest engagement, even with lower traffic. Window displays need optimization despite high foot traffic.
